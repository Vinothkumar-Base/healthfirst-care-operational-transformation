# Data Analysis & Dashboard

This section summarizes operational data analysis performed on appointment, resource, and patient feedback datasets.

## Analysis Performed
- Data cleaning and standardization
- Pivot-based utilization analysis
- Department-level workload comparison
- Peak appointment hour identification
- Patient satisfaction trend analysis

## Key Insights
- Peak appointment demand identified during mid-day hours
- Resource utilization variability across departments
- Manual booking inefficiencies impacting processing time
- Correlation between scheduling delays and satisfaction scores

Analysis was performed using Excel (data cleaning, pivot tables, interactive dashboards).

The dashboard was positioned as a decision-support tool for monitoring resource efficiency and operational bottlenecks.
